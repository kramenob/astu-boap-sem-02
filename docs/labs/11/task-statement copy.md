
# Лабораторная работа №11

## Задание:

Необходимо реализовать программный код, позволяющий найти сумму двух дробей, результат представить в виде несократимой дроби. При выполнении лабораторной работы следует: 
 - [X] T.1. Написать функцию, обеспечивающую ввод дроби с клавиатуры и проверку на корректность;
 - [ ] T.2. Написать функцию, возвращающую наибольший общий делитель двух натуральных чисел;
 - [ ] T.3. Написать функцию сокращения дроби;
 - [ ] T.4. Написать функцию, вычисляющую сумму двух дробей, Возвращаемая дробь должна быть несократимой;
 - [ ] T.5. Написать функцию подсчитывающую количество цифр в числе;
 - [ ] T.6. Написать функцию вывода дроби на экран (длина черты между числителем и знаменателем должна быть равна количеству цифр большего из выводимых чисел);
 - [ ] T.7. Вызывая функцию нахождения суммы дробей нужное количество раз, вывести результаты на экран.

## Индивидуальное задание:

Варианты с нечётными номерами необходимо решать следующим образом: вычислить сумму первых двух слагаемых как сумму обычных дробей; при необходимости полученную дробь сократить; до тех пор, пока не вычислена вся сумма, прибавлять к имеющейся дроби очередное слагаемое и при необходимости сокращать. Варианты с чётными номерами необходимо решать следующим образом:
 - [ ] I.1. определить числитель и знаменатель начального элемента последовательности 
 - [ ] I.2. вычислить очередной элемент как сумму обычных дробей; 
 - [ ] I.3. при необходимости полученную дробь сократить; 
 - [ ] I.4. повторять предыдущие пункты до тех пор, пока не будет получен требуемый элемент. При выполнении индивидуального задания следует:
	- [ ] I.4.1. Написать функцию, обеспечивающую ввод данных с клавиатуры и их проверку на корректность; тип и диапазон значений определить самостоятельно. 
	- [ ] I.4.2. Написать функцию нахождения суммы дробей нужное количество раз.

`Вариант 3`:
 * Для последовательности $а_1 = 1, a_{n+1} ={\frac {a_n + 1} {(1 + a_n)} }$ составить программу печати $n{\text{-го}}$ элемента в в виде несократимой дроби.
