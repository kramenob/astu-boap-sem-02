## 1. Постановка задачи

Необходимо разработать программу для формирования одномерного массива целых чисел по выбору пользователя (случайные числа или ручной ввод), а затем вычислить:

1. Произведение элементов массива с четными номерами.
2. Сумму элементов массива, расположенных между первым и последним нулевыми элементами.

---

## 2. Входные данные

- Количество элементов массива `N` (целое число, N > 0)
- Диапазон значений для случайной генерации (при выборе случайных чисел) — `minVal` и `maxVal`
- Элементы массива (при ручном вводе)
- Выбор способа формирования массива (`1` — случайные числа, `2` — ввод с клавиатуры)

---

## 3. Выходные данные

- Массив, сформированный пользователем
- Произведение элементов массива с четными индексами
- Сумма элементов массива между первым и последним нулевыми элементами (или сообщение о недостаточном количестве нулей)

---

## 4. Математическая модель задачи

Пусть массив A состоит из N целых элементов:  

```
A = [a0, a1, a2, ..., aN-1]
```

Требуется вычислить:

- Произведение элементов с четными индексами:

```
P = a0 * a2 * a4 * ... * a_{2k}, где 2k < N
```

- Сумму элементов между первым и последним нулями:

```
S = a_{firstZero+1} + a_{firstZero+2} + ... + a_{lastZero-1}
```

Если массив не содержит двух или более нулей, выводится соответствующее сообщение.

  void calcProductEvenIndexes(const vector<int>& arr) {
    long long product = 1;

    for (size_t i = 0; i < arr.size(); i += 2) {
      product *= arr[i];
    }

    cout << "Произведение элементов с четными номерами: " << product << endl;
  }

  void calcSumBetweenZeros(const vector<int>& arr) {

    int firstZero = -1;
    int lastZero  = -1;

    for (size_t i = 0; i < arr.size(); ++i) {
      if (arr[i] == 0) {
        if (firstZero == -1) {
          firstZero = static_cast<int>(i);
        }
        lastZero = static_cast<int>(i);
      }
    }

    int sumBetweenZeros = 0;

    if (firstZero == -1 || lastZero == -1 || firstZero == lastZero) {
      cout << "В массиве недостаточно нулевых элементов для вычисления суммы." << endl;
      return;
    }

    for (size_t i = firstZero + 1; i < lastZero; ++i) {
      sumBetweenZeros += arr[i];
    }

    cout << "Сумма элементов между первым и последним нулем: "
         << sumBetweenZeros << endl;
  }
